# CubeX

Welcome to the official documentation for **CubeX**, a high-performance, interactive visualization and analysis suite for radio astronomy spectral data cubes. Built on PyQtGraph, NumPy, and Astropy, CubeX is designed to deliver seamless exploration of FITS data with mathematically rigorous spatial and spectral analytics, along with real-time moment map generation, interactive Position-Velocity (PV) slices, and integrated Splatalogue line querying, and much more.
